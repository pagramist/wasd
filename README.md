# Book Tracker — трекер прочитанных книг

**Автор:** Попков Арсений

## Описание

GUI-приложение для учёта прочитанных книг. Позволяет добавлять книги, фильтровать по жанру и количеству страниц, сохранять данные в JSON.

## Технологии

- Python 3
- Tkinter (встроенный GUI)
- JSON для хранения

## Инструкция по запуску

1. Убедитесь, что установлен Python 3.
2. Скачайте файл `book_tracker.py` и поместите в любую папку.
3. Запустите программу:
   ```bash
   python book_tracker.py
